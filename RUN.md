# How to Run This Project

## Prerequisites

- **Node.js** 18+ ([nodejs.org](https://nodejs.org))

---

## Run

```bash
# 1. Go to project folder
cd MCP

# 2. Install dependencies (first time only)
npm install
cd client
npm install
cd ..

# 3. Start
npm run demo
```

Open **http://localhost:5173** in your browser.

---

## PowerShell (Windows)

If `&&` doesn't work, use `;`:

```powershell
cd MCP
npm install
cd client; npm install; cd ..
npm run demo
```
